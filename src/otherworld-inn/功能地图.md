# 异世旅社功能地图

这个文件列出当前项目中每个主要文件的位置和作用，方便维护、开源说明和二次开发。

## 总览

| 区域 | 说明 |
|---|---|
| `public/` | 人类玩家使用的网页前端 |
| `server/` | 游戏后端、世界生成、会话管理、API 路由 |
| `server/prompts/` | 所有可编辑提示词和主题规则 |
| `cli.js` | AI 玩家使用的命令行端 |
| `mcp-server.js` | MCP-to-CLI 透传桥，让 AI 通过 MCP 使用 CLI |

## 根目录文件

| 文件 | 作用 |
|---|---|
| `README.md` | 项目说明、启动方式、MCP 使用方式、API 配置说明 |
| `package.json` | 项目元信息、依赖、脚本：`npm start` / `npm run mcp` / `npm test` |
| `package-lock.json` | npm 依赖锁定文件 |
| `.gitignore` | 忽略本地密钥、运行数据、`node_modules`、日志等 |
| `.env.example` | 环境变量示例，主要用于 API key 配置 |
| `LICENSE` | 开源许可证 |
| `cli.js` | 玩家B 使用的命令行入口，也是 MCP 透传的目标 |
| `mcp-server.js` | MCP-to-CLI 透传桥，只提供 `output` 和 `input` 两个工具 |
| `功能地图.md` | 当前文件，说明项目文件结构和功能位置 |

## 前端文件

| 文件 | 作用 |
|---|---|
| `public/index.html` | 网页主界面结构，人类玩家端 |
| `public/app.js` | 前端主要逻辑：主题选择、开局、聊天、隐藏/公开切换、状态刷新、设置面板 |
| `public/style.css` | 网页 UI 样式 |
| `public/guide.html` | 独立玩法指南页面 |

## 后端文件

| 文件 | 作用 |
|---|---|
| `server/index.js` | Express 服务入口，挂载静态网页和 API 路由 |
| `server/config.example.json` | 默认示例配置，不含真实 key |
| `server/routes/rp-api.js` | 后端 API 路由：创建游戏、读取视角、发送行动、归档、提示词管理、配置管理 |
| `server/lib/ds-api.js` | DeepSeek 调用封装，负责请求模型 |
| `server/lib/rp-engine.js` | 核心规则：读配置、骰子、事件判定、创建 session |
| `server/lib/session.js` | 会话保存、读取、列表、归档、从归档恢复度假模式 |
| `server/lib/prompts.js` | 提示词加载、保存、拼装世界生成/公屏回复/隐藏判定 prompt |
| `server/lib/world-gen.js` | 开局世界生成流程：掷骰、生成副稿、解析角色、初始化世界 |
| `server/lib/world-workflow.js` | 每轮玩家行动处理：公屏回复、隐藏判定、分发给 A/B、事件检查、写日志 |
| `server/lib/time-tracker.js` | 游戏时间推进、天气生成、状态/背包追踪 |
| `server/lib/visibility.js` | 信息可见度：过滤公屏设定、组装 A/B 各自视角 |

## 提示词文件

| 文件 | 作用 |
|---|---|
| `server/prompts/player-base.json` | 玩家A/玩家B 的基础设定占位，可自行改成公开资料 |
| `server/prompts/world-gen.txt` | 世界生成总提示词 |
| `server/prompts/world-pool.txt` | 世界观素材池 |
| `server/prompts/world-maintain.txt` | 世界维持规则 |
| `server/prompts/public-reply.txt` | 公屏回复规则：只写环境，不替玩家行动/说话 |
| `server/prompts/hidden-judge.txt` | 隐藏行动三层判定规则 |
| `server/prompts/hidden-reply.txt` | 隐藏回复/渗透信息规则 |
| `server/prompts/theme-休闲.txt` | 休闲主题风格规则 |
| `server/prompts/theme-刺激.txt` | 刺激主题风格规则 |
| `server/prompts/theme-恐怖.txt` | 恐怖主题风格规则 |
| `server/prompts/theme-搞怪.txt` | 搞怪主题风格规则 |
| `server/prompts/theme-梦幻.txt` | 梦幻主题风格规则 |
| `server/prompts/theme-风格化.txt` | 风格化主题规则和作品内核卡 |

## 运行链路

```text
人类玩家
  -> public/index.html
  -> public/app.js
  -> server/routes/rp-api.js

AI 玩家
  -> mcp-server.js
  -> cli.js
  -> server/routes/rp-api.js

游戏后端
  -> server/lib/world-gen.js
  -> server/lib/world-workflow.js
  -> server/lib/session.js
  -> server/prompts/*
```

## 常用命令

```bash
npm install
npm start

npm test
```
